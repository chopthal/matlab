초기 실행

데이터 추가 후

Menu

File

  • Save Project : 현재 프로젝트를 저장합니다. 불러온 sensorgram curve들과 작업한 테이블 내용이 저장됩니다.
  • Load Project : 저장된 프로젝트를 불러옵니다.
  • Exit : 프로그램을 종료합니다.

Data

  • Add
    • iMSPR-mini Data : iMSPR-mini로 실험한 결과를 불러옵니다. 이 경우 Target, Reference, Log 3개의 파일을 각각 선택해야 합니다.
    • iMSPR-mini Data를 선택한 경우, Next 버튼을 누르면 Data Processing 창이 나타납니다. (Data Processing 항목 설명 참조)
    • iMSPR-Pro Data : iMSPR-Pro로 실험한 결과를 불러옵니다. 이 경우 실험 결과와 Info.mat 파일이 있는 부모 폴더를 선택해야 합니다.
    • Biacore Data : Biacore 장치로 실험한 결과를 불러옵니다. ‘.ble’ 파일은 불러올 수 없으며, referencing 된 결과를 하나의 txt 파일로 저장한 후 불러올 수 있습니다.
  • Export
    • Display Sensorgram(s) : 우측 테이블에서 display 체크 박스가 활성화 된 sensorgram curve 데이터를 내보낼 수 있으며, 형식은 ‘.txt’ 파일입니다.
    • All Sensorgrams : 모든 sensorgram curve 데이터를 내보냅니다.

About

  • Help : 도움말 페이지를 실행합니다.
  • Version : 현재 소프트웨어 버전을 보여줍니다.
  • Website : icluebio 웹사이트로 연결됩니다.

Main

Base line

  • Base line 기준 점 (시간)을 결정할 수 있습니다.
  • 모든 sensorgram curve들이 y축 방향으로 평행이동 하여 입력된 시점의 y값을 0으로 일치시킵니다.

Referencing

  • None : Referencing을 수행하지 않습니다.
  • N-T-N-T
    • Table을 기준으로 샘플의 배치 순서가 Negative - target - negative - target - ... 인 경우에 선택하세요.
    • 첫 번째 샘플은 반드시 Negative여야 합니다.
  • T-N-T-N
    • Table을 기준으로 샘플의 배치 순서가 Target - negative - target - negative - ... 인 경우에 선택하세요.
    • 마지막 샘플은 반드시 Negative여야 합니다.

Sensorgram

  • Display 항목이 체크된 sensorgram curve들을 표시합니다.

Start/End Point

  • Start Point : 결과를 분석하기 위한 초기 위치 를 설정합니다.
  • End Point : 결과를 분석하기 위한 마지막 위치 를 설정합니다.

Method

  • ΔRU (Yend - Ystart) : 마지막 위치 (End point)의 Y값 (Yend) 에서 초기 위치 (Start point)의 Y값 (Ystart) 을 뺀 결과를 표시합니다.
  • Averaging (Ystart - Yend) : 초기 위치에서 마지막 위치 까지의 Y값들의 평균 값을 표시합니다.
  • Drift (Linear fit) : 초기 위치와 마지막 위치의 Y값을 기준으로 선형 fitting을 수행하여 기울기 값을 표시합니다.

Correction

  • Screening 결과들의 편차를 Positive 항목을 기준으로 보정합니다.
  • None : 결과 보정을 수행하지 않습니다.
  • P-T-P-T
    • Table을 기준으로 샘플의 배치 순서가 Positive - target - positive - target - ... 인 경우에 선택하세요.
    • 첫 번째 샘플은 반드시 Positive여야 합니다.
  • T-P-T-P
    • Table을 기준으로 샘플의 배치 순서가 Target - positive - target - positive - ... 인 경우에 선택하세요.
    • 마지막 샘플은 반드시 Positive여야 합니다.

Preview

  • >> (Run) : Start/end point, Method, Correction 설정을 반영하여 결과를 분석합니다.
  • Detailed view : 분석된 결과를 detailed view 창에서 자세히 확인 할 수 있습니다.

Table

  • Sort (1, 2, 3) : Index 순서로 항목들을 재배치합니다.
  • Delete : 선택된 항목을 삭제합니다.
  • Check : Display 열의 체크 박스를 일괄 선택 또는 해제 합니다.
  • Table Contents
    • Index
      • 항목의 순서를 결정합니다. 해당 항목을 더블 클릭하여 수정 가능합니다.
      • 하나의 index 셀을 선택한 후 마우스 우클릭 - Numbering index from selected row를 선택하면 선택된 셀의 index를 시작으로 하위 셀 index 숫자 자동 채우기를 수행합니다.
    • ID : Type이 Target인 항목의 순서를 자동으로 결정합니다. 이 항목은 수정 불가능하며 Index, Sort, Type 등의 기능을 수행하면 자동으로 채워집니다.
    • Type : 선택된 항목의 속성을 결정합니다. Target, Positive, Negative중 하나를 선택할 수 있습니다.
    • Display : 현재 항목이 좌측 Sensorgram 및 Preview 축에 보여질 지 여부를 선택합니다.

Detailed view

  • Axes : Main window에서 얻은 결과들이 표시됩니다.
  • 가로 방향의 dotted-line은 설정된 threshold 값을 표시합니다.
  • 우측 하단의 table에는 threshold 이상의 값을 갖는 결과 항목들을 표시합니다.

Marker

  • Marker size : axes에 표시되는 마커의 크기를 입력합니다.
  • Marker : 마커의 색상을 결정합니다.
  • Positive : Positive 결과의 마커 색상을 결정합니다.
  • Negative : Negative 결과의 마커 색상을 결정합니다.
  • Threshold : Threshold 이상의 값을 가지는 결과의 마커 색상을 결정합니다.

Threshold

  • RU : Threshold 기준 값 (Y값, RU)을 설정합니다.
  • Axes에 표시되는 점선의 색상을 설정합니다.

Export Data

  • 현재 테이블에 표시된 값들을 ‘.txt’파일로 내보냅니다.

Data Processing

1st Injection start point (s)

  • Target : Target 데이터의 첫번째 주입 시점을 입력합니다.
  • Ref : Reference 데이터의 첫번째 주입 시점을 입력합니다.

X-shift to the left (s)

  • 1st injection start point 항목에서 결정된 주입 시간 범위를 그래프 상의 왼쪽으로 이동시킬 시간을 입력합니다.
  • 1st injection start point 를 설정하여 결과를 관찰한 다음 Base line을 잡을 수 있는 범위만큼 설정해 주는 것이 좋습니다.
  • 아래의 예시처럼 Target = 1989, Ref = 1981을 입력한 후 Apply 버튼을 눌러 injection timing을 결정하였을 때, 설정된 Injection 영역이 실제 영역을 벗어나는 경우가 발생할 수 있습니다.
  • 이 때는 아래와 같이 X-shift add 값을 추가하여 범위를 좌측으로 이동할 수 있습니다.

Buttons

  • Apply : Add, Target, Ref 설정 값을 반영하여 평행 이동 및 referencing을 수행합니다.
  • Reset : 평행 이동 및 referencing 이전으로 결과를 되돌립니다.
  • Next : 다음 단계로 진행합니다.
  • Cancel : Data processing 과정을 취소합니다.
